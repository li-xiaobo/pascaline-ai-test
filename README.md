# Pascaline Test: 8 AI Models vs. a 4-Year-Old's Toy

This repository archives my experiment testing 8 major AI models on spatial intelligence using a children's Pascaline toy.

## Key Finding
None of the 8 models understood the two-layer carry gear design: 10 teeth on the bottom, 1 tooth on top.
